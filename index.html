html

웹 표준과 웹 접근성을 준수하는 html 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 특히 웹표준을 위해 html5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다.

css

css 코딩은 IE를 지원도 충분히 고려하는 cross browsing 을 위해 안정적인 css 속성을 사용해서 html elememts를 배치하고 디자인합니다. 특별한 경우 css3에서 새롭게 지원하는 속성을 활용해서 좀 더 다양한 css 배치와 디자인을 합니다.

반응형 웹 제작에 있어서 cross browsing을 위해 안정적인기존 방식인 float,position,media query를 활용한 반응형웹 제작 경험을 충분히 가지고 있습니다.

주로 flex로 제작하며 gird 방식으로 반응형 웹사이트 제작도 충분히 가능합니다.

jquery / javascript

네비게이션,텝메뉴,어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작하며, 간단한 인터랙션 또한 제이쿼리와 자바스크립트로 표현 가능합니다.

오픈 소스와 플러그인의 경우 이해하고 수정하여 원하는 페이지를 구현할 수 있습니다.
